## Build/Install
**For Minilibx Library**

```$ make```

**For non MLX users (PPM version)**

```$ make -e ppm=1```

## Usage

```$ ./miniRT [filename.rt]```

- Plenty of scenes are present to test the program in ```./scenes``` folder.
- image.ppm is created when using PPM version.
